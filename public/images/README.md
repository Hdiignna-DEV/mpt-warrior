# Images Folder

## Founder Photo
**File:** `founder-photo.jpg`
**Lokasi:** `/public/images/founder-photo.jpg`

### Spesifikasi:
- **Ukuran:** 500x500px (square/kotak)
- **Format:** JPG atau PNG
- **Ukuran file:** Max 500KB (untuk loading cepat)
- **Background:** Netral atau profesional
- **Style:** Foto close-up wajah/headshot profesional

### Cara Upload:
1. Siapkan foto profesional Anda
2. Resize ke 500x500px menggunakan tool seperti:
   - Canva
   - Photoshop
   - Online image resizer
3. Save as `founder-photo.jpg`
4. Copy file ke folder `public/images/`
5. Refresh browser untuk melihat hasilnya
