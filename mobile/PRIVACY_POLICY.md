# Privacy Policy - MPT Warrior Mobile App

**Last Updated**: January 10, 2026

## 1. Overview
MPT Warrior ("we," "our," or "us") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, disclose, and otherwise handle your information when you use our mobile application.

## 2. Information We Collect

### User Account Information
- Email address
- User name and profile data
- Password (encrypted)
- Subscription status

### Trading Data
- Trading journal entries
- Trade history
- Account performance metrics
- User preferences and settings

### Device Information
- Device type and OS version
- Device ID
- Push notification tokens
- App installation and usage data

### Automatically Collected Data
- IP address
- App crash reports
- Performance metrics
- Analytics data

## 3. How We Use Your Information

We use collected information for:
- Providing and improving the app functionality
- Sending push notifications about trading alerts and updates
- Authenticating users and securing accounts
- Analyzing app usage and performance
- Responding to user requests and support inquiries
- Compliance with legal obligations

## 4. Data Storage and Security

- User data is stored securely using encrypted protocols
- We use JWT tokens for authentication
- API communications are encrypted via HTTPS
- Personal data is never shared with third parties without consent

## 5. Third-Party Services

The app uses the following third-party services:
- **Azure Cosmos DB**: For secure data storage
- **Expo Push Notifications**: For push notification delivery
- **Google Play & Apple App Store**: For app distribution

## 6. Push Notifications

We send push notifications for:
- Trading alerts and updates
- Account notifications
- Important app updates

You can disable notifications in app settings at any time.

## 7. User Rights

You have the right to:
- Access your personal data
- Request data correction
- Request data deletion
- Disable push notifications
- Export your trading data

## 8. Data Retention

- Account data is retained for as long as your account is active
- Deleted accounts are purged within 30 days
- Transaction logs are retained for 90 days for compliance

## 9. Children's Privacy

This app is not intended for users under 13. We do not knowingly collect data from children under 13. If we discover such data, we will delete it immediately.

## 10. Policy Changes

We may update this policy periodically. Continued use of the app after changes constitutes acceptance of the new terms.

For privacy concerns or data requests:
- Email: privacy@dedendev.com
- Support: support@mpt-warrior.app
