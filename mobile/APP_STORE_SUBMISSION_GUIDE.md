## üì¶ Build & Submit Process

### Step 1: Setup EAS CLI
```bash
cd mobile
npm install -g eas-cli
eas login
eas whoami
```

### Step 2: Configure Credentials
```bash
# For iOS
eas credentials --platform ios

# For Android
eas credentials --platform android
```

### Step 3: Build for App Stores
```bash
# iOS
eas build --platform ios --auto-submit

# Android
eas build --platform android --auto-submit
```

### Step 4: Create App Store Listings

**For iOS:**
1. Go to [App Store Connect](https://appstoreconnect.apple.com)
2. Click "My Apps" ‚Üí "Create"
3. Fill in app name, bundle ID, platform
4. Complete app information:
   - Category: Finance
   - Privacy policy URL
   - Support URL
   - App description & screenshots
5. Fill in Pricing & Availability
6. Submit for review

**For Android:**
1. Go to [Google Play Console](https://play.google.com/console)
2. Click "Create app"
3. Fill in basic info
4. Complete store listing:
   - App name, description, screenshots
   - Category, rating, content rating
   - Privacy policy URL
5. Complete content rating questionnaire
6. Set pricing (free/paid)
7. Submit for review

### Step 5: Monitor Reviews
- iOS: Usually 24-48 hours review time
- Android: Usually 2-4 hours review time
- Address any rejection feedback
- Resubmit with fixes
